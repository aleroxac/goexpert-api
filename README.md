# goexpert-api
API criada durante modulo de APIs do treinamento GoExpert da FullCycle



## Como executar o projeto
``` shell
make install
make run
```



## Funcionalidades
- [x] Router: go-chi
- [x] Middlewares: LogRequest(custom), Logger(go-chi), Verifier(go-chi), Authenticator(go-chi), Recoverer(go-chi)
- [x] Database: sqlite
- [x] Auth: jwt
- [x] Tests: unit-tests
- [x] Docs: swag, swagger
